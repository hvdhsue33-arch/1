# 🚀 دليل التشغيل السريع - نظام إبراهيم للمحاسبة

## ⚡ التشغيل السريع

### 1. **تثبيت التبعيات**
```bash
npm install
```

### 2. **تشغيل المشروع**
```bash
npm run dev
```

### 3. **فتح المتصفح**
اذهب إلى: `http://localhost:5000`

## 🛠️ التقنيات المستخدمة

### Backend
- **Node.js** + **Express.js**
- **MongoDB** + **Mongoose**
- **bcrypt** للتشفير
- **express-session** للجلسات

### Frontend
- **React** + **TypeScript**
- **TailwindCSS** للتصميم
- **Framer Motion** للحركة
- **shadcn/ui** للمكونات
